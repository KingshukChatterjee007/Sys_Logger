# Sys_Logger Client - PC

This installer is pre-configured for node: PC
Organization: Piyush Client Org

## Windows Setup
1. Extract this ZIP to a folder (e.g. C:\SysLogger)
2. Right-click setup_windows.ps1 and select "Run with PowerShell" (as Administrator)
3. The client will install silently and start on every boot.

No additional configuration is needed — this package is already set up for your node.
